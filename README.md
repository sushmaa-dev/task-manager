# ✅ Task Manager

A clean and minimal task management app built with React. Manage your daily tasks with an interface that remembers your tasks between sessions.

Coming soon 

## ✨ Features
- ➕ Add, complete, and delete tasks with a clean minimal interface
- 🔍 Filter tasks by All, Active, and Completed with real-time updates
- 💾 Tasks automatically saved to localStorage — no data loss on refresh
- 💡 Smart empty states that change based on the active filter
- ⌨️ Keyboard support — press Enter to quickly add tasks
- 📱 Responsive design optimized for both mobile and desktop

## 🛠 Tech Stack
- React
- Vite
- CSS
- uuid

## 🚀 Getting Started

**Prerequisites**
- Node.js

1. Clone the repo `git clone <your-repo-url>`
2. Install dependencies `npm install`
3. Run the app `npm run dev`
